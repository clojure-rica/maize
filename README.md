# Maize

Generates and solves random mazes using a depth first search.

![](images/example-maze.gif)

## Usage

Grab the code:

```bash
$ git clone https://github.com/clojure-rica/maize.git
$ cd maize
```

Compile it:

```bash
$ lein cljsbuild once
```

And take a peek:

```bash
$ open site/index.html
```
